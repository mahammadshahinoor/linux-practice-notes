# Linux Practice Notes

## File Commands
- ls → list files
- cd → change directory
- pwd → print working directory
- mkdir → create directory
- rm → remove file

## User Management
- useradd username
- passwd username
- groupadd groupname
- chown user:file

## Process Management
- ps aux
- top
- kill PID
- systemctl status nginx

## Disk Management
- df -h
- du -sh
